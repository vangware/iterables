Function some

  • some<Item, Predicated>(predicate: Predicate<Item, Predicated>): ((iterable: AsynchronousIterable<Item>) => Promise<boolean>)

  • Evaluates items in an iterable against a predicate and returns true if any item evaluates to true.

    Example

    const someEven = some((number: number) => number % 2 === 0);
    await someEven([1, 2, 3, 4]); // true
    await someEven([1, 3, 5, 7]); // false

    Returns

    Curried function with predicate set in context.

    Parameters
